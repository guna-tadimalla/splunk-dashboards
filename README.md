# Splunk Dashboards

## Overview
A collection of Splunk dashboards and SPL queries built while 
learning SIEM skills for SOC analyst work.

## Dashboards

### 1. Failed Login Detection
Detects brute force attempts by monitoring Windows Event ID 4625.
- Status: ⏳ Planned
- ATT&CK Technique: T1110 - Brute Force

### 2. Suspicious Process Creation
Monitors for unusual parent-child process relationships.
- Status: ⏳ Planned
- ATT&CK Technique: T1059 - Command and Scripting Interpreter

### 3. DNS Anomaly Detection
Identifies unusual DNS queries that may indicate C2 communication.
- Status: ⏳ Planned
- ATT&CK Technique: T1071 - Application Layer Protocol
